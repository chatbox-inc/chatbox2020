## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## エラーが起きた時

eslint errorが起きた場合、以下のコマンドを実行する事で修正できます。  
(赤文字の黒画面）
```bash
$npm run lintfix:js
```
